How to publish on GitHub Pages

  1. Create a new public repo (e.g., lightning-out-demo) on GitHub.
  2. Place this file at the repo root and name it index.html (you can copy from this host/index.html).
  3. Commit & push to the main branch.
  4. Repo Settings > Pages:
    • Source: "Deploy from a branch"
    • Branch: main (root)
  5. After Pages is active, note the URL: https://your-username.github.io/lightning-out-demo/
  6. Set your Connected App Callback URL to the full page URL:
    https://your-username.github.io/lightning-out-demo/index.html
  7. In this file, set:
    • CLIENT_ID to your Connected App Consumer Key
    • REDIRECT_URI to your Pages URL (exactly the same as Callback URL)
    • Keep SF_DOMAIN as your My Domain URL
  8. Commit & push the updated index.html.
  9. Open your Pages URL, click "Login to Salesforce", finish OAuth, and the LWC will render.
